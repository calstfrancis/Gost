# Bundled Fonts

Place the GOST Type B font file here as `gost-type-b.ttf` (or any `.ttf`/`.otf` file).

## Getting GOST Type B

GOST Type B implements the USSR/Russian engineering drawing lettering standard
(GOST 2.304-81). Free implementations:

- Search "GOST type B" on your system package manager
- GitHub: search "gost2304 font" or "GOST type B ttf"
- On openSUSE: may be available in the `terminus-bitmap-fonts` or similar packages

The font file name does not matter — all `.ttf` and `.otf` files in this directory
are loaded automatically.
